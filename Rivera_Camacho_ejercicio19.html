JavaScript Output

Ejemplos de las formas de mostrar datos en JavaScript.

1. innerHTML

Cambia el contenido HTML de un elemento. Puede incluir etiquetas como <b>.

Aquí aparecerá el resultado

2. innerText

Cambia solo el texto de un elemento. No interpreta etiquetas HTML.

Aquí aparecerá el resultado

3. document.write()

Escribe en el documento. Si se llama después de cargar la página, borra todo el HTML.

Aquí aparecerá el resultado (simulado)

4. window.alert()

Muestra un cuadro emergente con un mensaje.

5. console.log()

Muestra datos en la consola del navegador. Abre con F12 → Console.

Presiona el botón y abre F12

6. window.print()

Abre el diálogo de impresión del navegador.
